# Epistemological Triage: A Bayesian Framework for Extraordinary Claims

## Status: READY FOR PUBLICATION

## Purpose
Develops a formal Bayesian triage matrix for evaluating extraordinary
claims using Type I/Type II error analysis, the Law of Epistemic Warrant,
and zero-trust evaluation protocols. Provides a discipline-agnostic
methodological framework for scientific evaluation of anomalous phenomena.
