# Chaos, Attractors, and Determinism: Local Stochasticity in Globally Deterministic Systems

## Status: QUEUED

## Purpose
Uses nonlinear dynamics, Lyapunov exponents, and strange attractor theory to
prove that a system can be simultaneously locally unpredictable and globally
deterministic. Resolves the philosophical tension between free will and
predestination through rigorous mathematics of dynamical systems.
